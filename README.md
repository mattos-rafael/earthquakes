# 🌍 Earthquake Watcher

A real-time earthquake monitoring web application that visualizes seismic activity worldwide using interactive maps. Users can track earthquakes, filter by magnitude and date, and save favorite earthquakes to a personalized list.

## ✨ Features

- **Interactive Maps**: Two map views using Leaflet.js for comprehensive earthquake visualization
- **Real-time Data**: Fetches live earthquake data from the USGS API
- **Advanced Filtering**:
  - Filter earthquakes by magnitude
  - Filter by date range
  - Sort and customize your view
- **User Authentication**: Secure login and registration system powered by Firebase
- **Favorites System**: Save earthquakes to your personal favorites tab
- **Favorite Filters**: Filter and organize your saved earthquakes
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## 🛠️ Technologies Used

### Frontend
- **HTML5/CSS3**: Structure and styling
- **JavaScript (ES6+)**: Core functionality
- **Leaflet.js**: Interactive mapping library
- **Fetch API**: Data fetching from USGS

### Backend & Services
- **Firebase Authentication**: User registration and login
- **Firebase Firestore**: Database for storing user favorites
- **USGS Earthquake API**: Real-time earthquake data
